HAP動画のサムネイルをFinderに自動生成するmacOSアプリ。

VJやプロジェクションマッピングで多用されるHAP動画は、macOSのFinderではサムネイルが表示されません。VJClipThumbGenはフォルダを登録するだけで、HAP動画にFinderカスタムアイコンを自動設定。QuickLookサムネイル拡張も同梱しており、Finderのプレビュー機能でもサムネイルが表示されます。

## 機能

- メニューバー常駐 — フォルダを登録するだけで自動処理
- Finderカスタムアイコン — HAP動画の内容がひと目でわかる
- QuickLookサムネイル — Finderのプレビューにも対応
- 高速処理 — 大量のクリップも効率的に処理

## 対応フォーマット

- HAP (Hap1 / DXT1)
- HAP Alpha (Hap5 / DXT5)
- HAP Q (HapY / YCoCg-DXT5)
- HAP Q Alpha (HapM)
- 対応ファイル形式: .mov / .avi

## 動作環境

- macOS 12.0 Monterey 以降
- Apple Silicon / Intel 両対応
